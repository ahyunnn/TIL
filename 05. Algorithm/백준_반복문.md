##### 백준 25304번 영수증

```python
total = int(input())
total_num = int(input())

cal_price = 0
for _ in range(total_num):
    price, num = list(map(int, input().split()))

    cal_price += (price*num)

if total == cal_price:
    print('Yes')
else:
    print('No')
```
